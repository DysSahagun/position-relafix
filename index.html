Position: Relafix

A positioned-fixed element which has a parent that can be positioned-absolute or floated.

The HTML markup

<div class="relafix">
    <div class="relafix-cr">
          Content
    </div>
</div>

The CSS

.relafix,
.relafix-cr {
	width: 15em;
}

.relafix {
	float: right;
}

.relafix-cr {
	position: fixed;
}

Browser support

Firefox, Chrome, Safari, IE 7-8
